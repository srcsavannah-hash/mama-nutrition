# 🚀 5分钟部署指南

## 你需要准备
- 一个 GitHub 账号（免费）
- 一个 Vercel 账号（免费，用 GitHub 登录）
- 一个 Anthropic API Key

---

## 第一步：获取 Anthropic API Key
1. 打开 https://console.anthropic.com
2. 注册账号（需要海外手机号或邮箱）
3. 进入 API Keys → Create Key
4. 复制保存这个 Key（格式类似 sk-ant-...）

---

## 第二步：上传代码到 GitHub
1. 打开 https://github.com → 注册/登录
2. 点击右上角 "+" → "New repository"
3. 名字填 `mama-nutrition`，点 Create
4. 把本文件夹所有文件上传进去（拖拽即可）

---

## 第三步：部署到 Vercel
1. 打开 https://vercel.com → 用 GitHub 登录
2. 点击 "Add New Project"
3. 选择刚才的 `mama-nutrition` 仓库
4. 点击 "Environment Variables"，添加：
   - Name: `ANTHROPIC_API_KEY`
   - Value: 粘贴你的 API Key
5. 点击 "Deploy"，等待约1分钟

---

## 第四步：发给妈妈！
部署完成后，Vercel 会给你一个链接，类似：
`https://mama-nutrition-xxx.vercel.app`

把这个链接发给妈妈，她用微信打开就能用了 🎉

### 妈妈怎么保存到桌面？
1. 微信打开链接
2. 右上角 "..." → 在浏览器打开
3. 浏览器右上角 "分享" → 添加到主屏幕
4. 桌面上会出现一个图标，跟 App 一样！

---

## 遇到问题？
- **部署失败**：检查 ANTHROPIC_API_KEY 是否正确填写
- **打开白屏**：清除浏览器缓存后重试
- **AI 不回复**：检查 API Key 是否有余额（新账号有免费额度）
